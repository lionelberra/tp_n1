ACT:

1. ¿cuales son los tipos de titulos que hay?

2. ¿cual es la importancia del numero que lleva la etiqueta de titulo?

3. ¿que es lo que debe evitarse para que el uso de los titulos sea accesible a las personas no videntes?. justificar

4. ¿cual es el concepto de anidar titulos? Ejemplificar con un ejemplo distinto al propuesto por MDN.

5. investiga que hacen las etiquetas spam, i, b, a.

6. en la estructura basica del documento HTML, ¿como definiria html - head - body

RESP:

1. los tipos de titulos son de h1 a h6. estos tienen una diferencia, cada "h" tiene su propio tamaño, siendo el h1 el mas grande y el h6 el mas chico.

2. la importancia del numero de la etiqueta es el tamaño que va a llevar

3. debe evitarse la falta de una etiquta, osea saltar una etiqueta como ejemplo, saltar de h1 a h3.

4. el concepto de anidar titulos se refiere a organizar los titulos de manera jerarquica y logica. por ejemplo

los hijos de los dos mejores jugadores del futbol:

lionel messi:

Ciro Messi

Mateo Messi

Thiago Messi

cristiano ronaldo:

Cristiano Júnior

los mellizos Eva y Mateo

Alana Martina

Bella Esmeralda

5.

span se ultliza para aplicar estilos a un fragmento de texto en linea.

i se utiliza para aplicar estilo en cursiva a un fragmento de texto.

b se utiliza para aplicar estilo en negrita a un fragmento de texto.

a se utiliza para crear enlaces a otras paginas web.

6.

html es la etiqueta raiz que encapsula todo el contenido del documento HTML.

head es una seccion dentro del documento que generalmente contiene informacion introductoria, como el titulo, logotipos o menus de navegacion.

body es la seccion principal del documento donde se coloca el contenido visible para el usuario, como texto, imagenes, videos, etc.
